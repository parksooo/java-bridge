```
## BridgeGame
- 입력 받은 문자 "U", "D" 에 따라서 움직일지 판단하는 기능
- 게임이 끝났을 때, 재시작 할지 묻는 기능
- 올바른 문자가 아닐 경우 재시작 할지 묻는 기능
```
```
##BridgeMaker
- 다리에 길이를 사용자에게 받고 그만큼 List<String>을 생성 하는 기능
- 리스트 안에는 랜덤 숫자에 비교하여, 0 -> "D", 1 -> "U"로 만드는 기능
- 랜덤 숫자는 인터페이스 BridgeNumberGenerator에서 상속 받은 기능을 사용하여서 구현
```
```

##InputView
- 입력 받은 다리 길이를 int 형으로 변화해주는 기능
- 입력 받은 "U", "D"를 리턴해주는 기능
- 사용자가 다시 게임을 할지에 대한 "Q", "R"을 입력 받는 기능
- 원하는 입력인지 판단하는 validator와 연동하여서 구현
```
```
##OutPutView
- 현재까지 입력받은 올바른 정답으로 move한 다리를 출력해주는 기능
- 모든 다리를 이동했다면, 결과를 출력하는 기능
- 중간에 오답을 입력했을 경우, 재시작 질문을 출력하는 기능
- 이동할 칸을 물어보는 출력 기능
- 총 시도한 횟수를 출력하는 기능
```
```
##Validator
- 3 ~ 20 사이의 다리길이를 입력했는지 판단하는 기능
- 위, 아래를 판단하는 "U", "D"를 입력했는지 판단하는 기능
- 재시작, 종료를 판단하는 "R", "Q"를 입력했는지 판단하는 기능
```
